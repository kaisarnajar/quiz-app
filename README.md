# Simple Quiz App

A simple and interactive quiz application built using HTML, CSS, and JavaScript. The app presents multiple-choice questions, accepts user answers, and displays the final score at the end.

## Features

* Multiple-choice questions
* Instant answer selection
* Score calculation
* Responsive and clean interface
* Built with vanilla JavaScript

## Project Structure

```text
simple-quiz-app/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## Getting Started

### Clone the repository

```bash
git clone <repository-url>
cd simple-quiz-app
```

### Run the project

Simply open `index.html` in your preferred web browser.

No additional setup or dependencies are required.

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Purpose

This project was built to practice JavaScript fundamentals, including DOM manipulation, event handling, arrays, objects, and application logic.
